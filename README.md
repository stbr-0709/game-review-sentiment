# 游戏评论情感分析器

AI驱动的游戏用户评论分析工具，支持批量分析、情绪判断、问题提取。

## 功能特性

- 🤖 AI智能分析（DeepSeek API）
- 📊 情绪分类（正面/负面/中性）
- 🔍 问题关键词提取
- 📁 支持批量导入（直接粘贴 / Excel/CSV）
- 💾 自动导出CSV结果

## 快速开始

1. 安装依赖：`pip install openai pandas openpyxl`
2. 配置API Key（DeepSeek）
3. 运行：`python review_api_batch.py`
